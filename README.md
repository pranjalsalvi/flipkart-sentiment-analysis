# Flipkart Product Review Sentiment Analysis

## Objective
To classify Flipkart product reviews as positive or negative using NLP and machine learning.

## Technologies Used
- Python
- NLP (NLTK, spaCy)
- TF-IDF
- SVM
- Streamlit

## Features
- Real-time sentiment prediction
- Pre-trained SVM model
- Streamlit web interface

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
